# Optimus Smart CRM

An AI-powered Customer Relationship Management (CRM) system built using Flask, Python, Machine Learning, and Data Analytics.

## Features

* Customer Management
* Lead Tracking and Organization
* Customer Segmentation using K-Means Clustering
* Lead Conversion Prediction using Logistic Regression
* Interactive Dashboard
* PDF Report Generation
* Search and Filter Functionality
* Modern Responsive UI

## Tech Stack

* Python
* Flask
* Pandas
* Scikit-Learn
* ReportLab
* HTML
* CSS
* JavaScript

## Machine Learning Features

### Customer Segmentation

Uses K-Means Clustering to group customers based on behavior and business metrics.

### Lead Prediction

Uses Logistic Regression to predict conversion probability and identify high-value leads.
